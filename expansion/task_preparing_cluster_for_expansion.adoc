= 準備叢集以進行擴充
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要為叢集做好擴充準備、您必須新增節點鎖定的授權、驗證系統健全狀況、備份叢集的組態、並產生AutoSupport 一則消息。
