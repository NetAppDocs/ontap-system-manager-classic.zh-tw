= 設定對等關係ONTAP （不含版本號的版本）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用ONTAP 包含ONTAP ESI9.2的《SVM系統管理程式_classic_》介面或較早ONTAP 版本的《SVM對等關係》、即可建立SVM對等關係。

對等關係可定義網路連線、讓叢集和SVM能夠安全地交換資料。您必須先建立叢集對等關係、才能建立SVM對等關係。
