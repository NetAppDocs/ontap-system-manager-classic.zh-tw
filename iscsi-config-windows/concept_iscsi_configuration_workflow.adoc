= iSCSI組態與資源配置工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您使用iSCSI將儲存設備提供給主機時、請在儲存虛擬機器（SVM）上配置一個Volume和LUN、然後從主機連線至LUN。
