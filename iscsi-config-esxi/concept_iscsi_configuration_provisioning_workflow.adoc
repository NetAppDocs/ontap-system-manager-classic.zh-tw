= iSCSI組態工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用iSCSI將儲存設備提供給ESXi主機時、您可以使用VMware vSphere的虛擬儲存主控台、在儲存虛擬機器（SVM）上配置磁碟區和LUN、然後從主機連線至LUN。
